
# Cricket Stuffs

## Description
A simple program to  calculate some cool informations of [cricket](https://www.google.com/search?q=cricket).

Can do calculate the following stuffs:

* Run Rate
* Current Run Rate
* Net Run Rate
* Required Run Rate
* Projected Score
* Strike Rate
* Bowler Average

## Getting Started
### Requirements
* PC
* Time
* Mood

### Installations
* Any C++ compiler
* Any IDE/IDLE

Just fire the file name
```
main.cpp
``` 
